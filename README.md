# UnityMap

UnityMap is a simple web project created for the Hack4Unity online hackathon on Devpost.

Its goal is to promote unity and community support by allowing users to add ‚ÄúHelp Points‚Äù on a map.  
Each Help Point represents someone offering support such as:
- food donations  
- tutoring  
- pet care  
- community activities  
- or any helpful act  

This project encourages a more unified and connected world.

---

## Features
- Add Help Points with a title and description  
- Display all community help markers  
- Simple and beginner-friendly code (HTML, CSS, JS)  
- Fully works without backend  

---

## How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser  
3. Add your help points and visualize them instantly

Even small actions make a huge difference when communities work together.
